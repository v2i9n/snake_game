# 🐍 Snake Game - لعبة الأفعى

لعبة الأفعى الكلاسيكية مبرمجة بـ Python و Pygame

### 🎮 طريقة اللعب
استخدم الأسهم لتحريك الأفعى وكل التفاح الأحمر. كل ما تاكل أكثر، تطول أكثر!

### 🚀 التشغيل
1. ثبت المكتبة: `pip install pygame`
2. شغل اللعبة: `python snake.py`

### ✨ المميزات
- عداد نقاط
- تزيد السرعة مع الوقت
- شاشة Game Over
- كود نظيف وسهل التعديل
